Pagina recopilatoria con las distintas etiquetas HTML usadas en las practicas

Bienvenido a mi primera pagina recopilatoria con distintas ejemplos practicos de como utilizar los comandos HTML
Si pinchas en cada uno de los enlaces te llevara a cada una de las practicas realizadas durante el curso monstrando un pequeño ejemplo de como usar cada comando.
A continuación se explicara brevemente la estructura básica de una página HTML y en la posterior tabla,se recogen los distintos enlaces a las prácticas realizadas. Espero, vayan ampliándose y mejorando el contenido de cada una.

Estructura básica página HTML (En tu código fuente, en lugar de las comillas "", deberas usar <> al abrir usar etiqueta-comando)

  • Cada página comienza con "HTML"
  • A continuación viene la cabecera, delimitada por "HEAD" y "/HEAD".
  • Despues el comando "BODY", que indica el comienzo y el cuerpo de la página. Las instrucciones HTML se escribiran a continuación, y finalizarán con "/BODY"
  • La página acabara con "/HTML".
